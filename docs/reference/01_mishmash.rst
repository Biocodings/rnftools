.. _mishmash:

MIShmash
========

Here you can find documentation for **all supported NGS read simulators**. Please note that you can use simplified names of classes like ``rnftools.mishmash.ArtIllumina`` (instead of longer ``rnftools.mishmash.artIllumina.ArtIllumina``). General information about read simulators can be found on a special page called :ref:`list-of-read-simulators`.
