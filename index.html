RNFtools

Introduction

RNF (Read Naming Format) is a generic format for assigning read names with encoded information about original positions. See its specification (its XeTeX/LaTeX source is available in a standalone GitHub repository).

RNFtools is an associated software package for RNF based on SnakeMake. A full documentation with tutorials is available on http://rnftools.rtfd.org. It works on any Unix-like OS with Python 3.2+. A preferred way of installation is using PIP:
> pip3 install rnftools
Source codes are available in its GitHub repository which can be cloned by
> git clone git://github.com/karel-brinda/rnftools
