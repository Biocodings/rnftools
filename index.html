RNFtools

RNFtools is an associated software package for Read Naming Format, a generic format for simulated NGS reads (see its specification). The whole package is based on SnakeMake. To install RNFtools, a Unix-like system with Python 3.2+ is required.

Installation

  • Using PIP:

    pip install rnftools

  • Using Easy Install:

    easy_install rnftools

Documentation

A full documentation with tutorials is available on http://rnftools.rtfd.org.

Publications
