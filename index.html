RNFtools

Introduction

RNF (Read Naming Format) is a generic format for assigning read names with encoded information about original positions. See its specification (its XeLaTeX source is available in a standalone GitHub repository).

RNFtools is an associated software package for RNF based on SnakeMake. A full documentation with tutorials is available on http://rnftools.rtfd.org. It can be easily installed using PIP by
> pip install rnftools
Source codes are available in its GitHub repository and can be cloned by
> git clone git://github.com/karel-brinda/rnftools
